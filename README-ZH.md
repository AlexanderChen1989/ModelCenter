## 安装

### 1. 用 pip 安装 (推荐)

```shell
$ pip install model-center
```

### 2. 从源代码安装

```shell
$ git clone https://github.com/OpenBMB/ModelCenter.git
$ cd ModelCenter
$ pip install -r requirements.txt
$ python3 setup.py install
```
